迭代器(Iterator)概述

  • Iterator是作为一个接口存在的，它定义了迭代器所具有的功能
  • 迭代器是一种设计模式，它是一个对象，它可以遍历并选择序列中的对象，而开发人员不需要了解该序列的底层结构;迭代器通常被称为“轻量级”对象，因为创建它的代价小.

迭代器的好处

  1. 迭代器可以提供统一的迭代方式。
  2. 迭代器也可以在对客户端透明的情况下，提供各种不同的迭代方式。
  3. 迭代器提供一种快速失败机制，防止多线程下迭代的不安全操作。
