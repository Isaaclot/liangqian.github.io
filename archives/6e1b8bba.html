Linux下挂载移动硬盘(ntfs格式)
发布于：2019-10-15 22:29:24 标签：/ Linux / Raspberry / 访问：

背景

  • 上手树莓派，打算将弃用的笔记本硬盘挂在树莓派主机上，作为下载机使用
  • 由于旧硬盘是win电脑拆下来的硬盘，磁盘格式为Microsoft basic data，也就是我们所说的NTFS格式，如果挂在Linux系统上，需要手动去挂载，并且防止出现文件乱码的情况
  • 查找资料发现linux支持ntfs格式分区需要安装ntfs-3g软件，安装使用即可

安装ntfs-3g

  1. 通过编译安装

    • 下载地址:https://www.tuxera.com/community/open-source-ntfs-3g/
    • 下载好了之后，解压文件夹
      • The latest stable version is ntfs-3g_ntfsprogs-2017.3.23, released on March 28, 2017.
    • 进入文件目录
      • ./configure
      • make
      • make install
    • 验证

  2. 使用Ubuntu的apt包管理器安装

    • 搜索
    • 安装
      • 选择安装ntfs-3g软件apt install ntfs-3g -y

        挂载

  3. 找到待挂载的硬盘(fdisk -l) 本页贴出自己挂载的硬盘信息(包括磁头、分区大小等信息)

  4. 建立加载点

  5. 将对应的磁盘分区挂载在建立的文件夹

  • 不关心是否出现乱码：mount –t ntfs-3g /dev/sdi1 /media
  • 防止出现乱码(注：对ntfs格式的磁盘分区应使用-t ntfs-3g参数，对fat32格式的磁盘分区应使用-t vfat参数。若汉字文件名显示为乱码或不显示，可以使用下面的命令格式)：mount -t ntfs-3g -o iocharset=cp936 /dev/sda2 /media/sda100

验证结果(挂载了三个盘)
